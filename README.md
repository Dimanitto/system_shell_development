# system_shell_development
Разработка системной оболочки C++
# Описание проекта
По ходу выполнений модулей попытался создать простейшую системную оболочку.
### Технологии
* C++
* Ubuntu 22.04
### Подготовка к запуску
- Скомпилируем .cpp файл
```
$ g++ -lm -o core core.cpp
```
- Запустим программу
```
./core
```
